# Acessibilidade do Site

Esta pasta contém recursos e arquivos dedicados a tornar o site mais acessível para todos os usuários.

## O que foi adicionado:
- Link "Pular para o conteúdo" no topo da página.
- Melhorias de contraste.
- Ajustes para navegação por teclado.
- Modo escuro acessível.
- Estrutura HTML mais semântica.

## Objetivo
Garantir que todas as pessoas, inclusive aquelas com deficiência ou limitações tecnológicas, consigam usar o site sem dificuldades.
